## 📌 Overview
ImpactSync is a highly responsive, zero-backend web application built to solve the **[Smart Resource Allocation] Data-Driven Volunteer Coordination for Social Impact** challenge. 

Traditional volunteer coordination often suffers from friction, relying on fragmented communication channels or static spreadsheets. ImpactSync provides local NGOs, community organizers, and volunteers with a seamless, centralized platform to post social impact needs and claim tasks in real-time. By streamlining the dispatch process, ImpactSync ensures that human resources are allocated efficiently where they are needed most.

## ✨ Key Features
* **Dynamic Resource Allocation:** Organizers can instantly post community needs, categorized by impact sector (Environment, Education, Health, Logistics) and estimated effort hours.
* **One-Click Claiming:** Volunteers can browse an active feed of "Open Needs" and claim them, instantly moving the task to the "Allocated" board.
* **Live Impact KPIs:** The dashboard automatically tracks and calculates vital metrics: Total Needs Posted, Tasks Claimed, and Total Impact Hours Allocated.
* **Zero-Backend Architecture:** Built using the browser's `localStorage` API, ensuring edge-level speed, zero server costs, and instant deployability.
* **Enterprise UI:** Styled with Tailwind CSS for a clean, accessible, and responsive user experience.

## 🛠️ Tech Stack
* **Frontend Structure:** HTML5
* **Styling:** Tailwind CSS (via CDN) & FontAwesome Icons
* **Logic & State Management:** Vanilla JavaScript (ES6)
* **Data Persistence:** Browser LocalStorage API
* **Hosting:** GitHub Pages
